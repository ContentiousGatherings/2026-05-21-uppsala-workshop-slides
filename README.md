# Developing a pipeline for extracting contentious gatherings from historical Swedish newspapers, 1820–1939

Slides for the talk presented at *From Archive to Dataset: Uppsala workshop on coding contention with new methods*, 2026-05-21.

**Authors:** Andrés Brink Pinto, Mathias Johansson, Magnus Olofsson (Lund University)

## View the slides

The rendered slides are published via GitHub Pages — see the repository's Pages URL.

To build locally:

```bash
quarto render index.qmd
open _site/index.html
```

Requires [Quarto](https://quarto.org) 1.5+.
